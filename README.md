## 🚀 Proyectos de Alto Impacto

Aquí presento algunas de las soluciones que he diseñado e implementado, enfocadas en la eficiencia y la regularización institucional.

* **SIRT-INTI:** Sistema de Regularización de Tierras. Implementado bajo estándares **Métrica V3** para la gestión nacional de tierras.
* **HMCP:** Sistema de Bienes Nacionales. Gestión hospitalaria crítica, optimizada con PHP, Apache y PostgreSQL.
* **Iglecia:** Solución de administración local. Desarrollo ágil con Python y Django enfocado en la usabilidad y manejo seguro de datos.
